# Portfolio — GBOYEE

Personal landing page showcasing projects and contact.

## Quick Deploy

Deploy to GitHub Pages:

1. Push this repo to `gboyee.github.io`
2. In repo Settings → Pages, set source to `main` branch `/ (root)`
3. Site goes live at https://gboyee.github.io/

## Structure

- `index.html` — single-page portable HTML (no build step)
- Customize the content to your taste.
