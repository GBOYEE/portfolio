# GBOYEE — AI Engineer Portfolio

Single‑page static portfolio showcasing AI/Backend engineering projects.

## 🚀 Quick Deploy

```bash
# Clone and open in browser
git clone https://github.com/GBOYEE/portfolio.git
cd portfolio
# Open index.html directly or serve:
python3 -m http.server 8000
# Then visit http://localhost:8000
```

## ✨ Features

- Cyberpunk‑inspired dark UI (Tailwind CSS)
- Responsive layout (mobile‑friendly)
- Typing animation on hero section
- Glassmorphism effects
- Mermaid diagrams embedded in project cards (coming soon)

## 📂 Structure

```
portfolio/
├── index.html      # Main page (self-contained)
├── css/            # Custom styles (optional)
├── js/             # Interactivity (optional)
└── projects/       # Individual project pages (future)
```

## 🔧 Customization

Edit `index.html` to update:
- Project links and descriptions
- Skills list
- Contact email

For GitHub Pages: enable Pages in repo settings, source: `main` branch, `/ (root)`.
